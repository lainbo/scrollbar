<h4 align="center">一个简单的在线滚动条设计工具</h4>

## 关于

Scrollbar.app 是一个简单的网页应用，帮助开发者设计和实现自定义滚动条。
这是[原项目](https://github.com/henripar/scrollbar)的中文汉化版本

## 项目设置

```sh
npm install
```

### 开发环境编译和热重载

```sh
npm run dev
```

### 生产环境编译和压缩

```sh
npm run build
```
